# Piggy Planner

Piggy Planner is a soft productivity dashboard for planning your day, tracking small health habits, and keeping an eye on spending.

## Quick Start

1. Open `index.html` in your browser.
2. Use the top navigation to move between `Home`, `Tracker`, and `Progress`.
3. Start on the Tracker page if you want to add tasks right away.

## What You Can Do

### Home

- See a quick snapshot of your day
- Swipe through simple reminders
- Jump into Schedule, Health, or Expense sections

### Tracker

- Add tasks with time and details
- Track habits such as water, steps, or sleep
- Record daily spending
- Remove items after adding them

### Progress

- Review daily summary cards
- Check your weekly rhythm in the chart
- Use the Pomodoro timer for a focused 25-minute session

## Files

- `index.html` - homepage
- `tracker.html` - daily tracking page
- `progress.html` - progress and focus page
- `scss/` - source styles
- `css/style.css` - compiled stylesheet used by the pages
- `js/` - interactions for carousel, tracker input, timer, and mobile nav
- `images/` - mascot and section illustrations

## Notes

- The layout is responsive and works on mobile, tablet, and desktop.
- Tracker items are handled in the browser for quick, lightweight interaction.
- If you change SCSS files, make sure your CSS output stays in sync with `css/style.css`.
